# Assignment2




The Life Hack Quiz App is an Android application that tests users on common life hacks and myths using a simple true/false quiz system.

s
- Welcome screen with start button
- Quiz screen with true/false questions
- Score tracking system
- Feedback after each answer
- Final score screen with personalised feedback
- Review button to retry quiz

s
1. User starts the app from the welcome screen
2. User answers multiple true/false questions
3. The app calculates the score
4. Final score is displayed with feedback message
5. User can review and retry the quiz

- Kotlin
- Android Studio
- XML Layouts
